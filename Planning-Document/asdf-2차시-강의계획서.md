# [asdf] 웹으로 만드는 나의 꿈 2차시 수업자료 초안

## 수업 목표
- 언플러그드 방식을 통한 HTML Tag와 Attribute 익히기
- 자신이 만들고 싶은 웹의 아이디어 구체화

## 교육 프롤로그
------------------

ex) 
[웹 코딩, 지금부터 시작이야!]

지금부터가 진짜 웹 코딩, 직접 웹을 만들어 보자!
